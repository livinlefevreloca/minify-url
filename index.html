Minify URL

A microservice by Adam Lefevre

How to use this App:

To retreive a shortened URL type "new" and the original url into the adress bar like so https://minifyurl.heroku.com/new/https://www.google.com

You will recieve a url like https://minifyurl.heroku.com/1234 which will take you to the original site.
